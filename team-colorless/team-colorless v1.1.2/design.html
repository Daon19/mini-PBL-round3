팀 포스터를 만들기 위한 디자인 자료조사

1. 디자인 자료조사 목적

  • 목적 : 포스터의 구성, 색감, 타이포그래피, 이미지 사용 방식 등을 분석
  • 활용 : 벤치마킹 + 우리 팀만의 방향성 도출

2. 자료조사 대상 추천 사이트

  • Canva : “청소년 팀 포스터”, “환경 캠페인”, “창업 아이디어 발표” 등
  • Pinterest : “team poster design”, “youth team identity”, “creative project poster”
  • 미리캔버스 : “포스터”, “청소년 캠페인”, “학급 포스터”
  • Behance : “Poster Design”, “Youth project branding”

3. 벤치마킹 워크시트

■ 포스터 명

어떤 포스터인가요? 예) 환경을 지키는 습관 캠페인

디자인 전시회 홍보 포스터 (미니멀리즘 테마)
-Gradual Color Reveal-
■ 색상

주로 사용된 색은? 예) 초록, 파랑

흑백(검정 배경 + 흰 텍스트)
하단에만 포인트 컬러로 보라색이 스며들듯 표현됨
■ 구성

어떤 레이아웃인가요? 예) 중앙 로고 + 하단 설명

중앙 상단: 큰 타이포그래피(이벤트 이름)
중앙 중간: 부제나 설명
하단 오른쪽: 포인트 컬러 그래픽 + 날짜 정보
■ 느낌/톤

전반적인 느낌은? 예) 활기찬, 명랑한

차분하고 세련됨, 동시에 색의 등장으로 ‘변화’와 ‘가능성’의 인상을 줌
■ 슬로건 위치

슬로건이 어디에 있는가요? 예) 상단 왼쪽

중앙 중간 정렬, 시선이 머무는 위치에 큼직하게 배치됨
■ 참고할 점

우리 팀이 참고하면 좋을 요소는? 예) 명확한 색감과 로고 강조

무채색 기반 위에 포인트 컬러를 ‘스며들듯’ 활용
팀의 색을 채워나간다는 메시지를 시각적으로 전달하는 방법
타이포 중심 레이아웃으로 이름과 컨셉 강조

4. 조사 후 할 일: '우리 팀의 디자인 방향 도출'

■ 어떤 포스터가 가장 눈에 띄었나요? 이유는?
Gradual Color Reveal 포스터
처음에는 완전히 흑백이지만, 하단에 점점 퍼지는 색이 인상적이었고,
‘무색’이라는 개념을 시각적으로 잘 표현하면서도,
변화(색이 생기는 과정)를 자연스럽게 전달해 주는 것이
우리 팀의 정체성과도 잘 맞는 느낌이 들었음.
■ 우리 팀 포스터에는 어떤 요소를 차용하고 싶은가요?
무채색 배경에 포인트 컬러만 사용하기
→ 우리 팀이 아직 ‘무색’이라는 걸 시각적으로 보여줄 수 있음
색이 스며드는 듯한 그래픽 효과
→ 앞으로 점점 팀의 색을 채워나갈 것이란 의미를 시각화
타이포 중심 디자인
→ ‘colorless’라는 팀 이름을 시각적으로 강조하는 데 효과적
■ 어떤 스타일은 우리 팀과 어울리지 않는다고 느꼈나요?
컬러풀하고 명랑한 분위기의 포스터
예: 팝 아트 스타일, 원색을 강하게 사용하는 포스터
이유: 우리 팀은 “지금은 아무 색도 없는 상태”에서 출발하기 때문에,
처음부터 너무 다양한 색이나 활기찬 느낌을 강조하면
팀의 정체성과 충돌할 수 있다고 느낌.
우리 팀은 좀 더 조용하고, 절제된 분위기에서
점차 변화하는 느낌이 더 잘 어울릴 것 같았음.
